## 1. Цель работы

Проверка корректности работы программы, которая:

– генерирует массив натуральных чисел размерности n;
– выводит массив в консоль;
– находит максимальную сумму цифр числа в десятичном представлении;
– выводит все числа, имеющие максимальную сумму цифр.

Функция `digit_sum()` возвращает сумму цифр числа в десятичном представлении.

---

## 2. Основные тесты

Программа запускалась с различными значениями размера массива.

### Тест 1

Входной параметр:
n = 10

Сгенерированный массив:
874 882 847 727 992 676 156 275 725 970

Контрольный расчёт сумм цифр:

874 → 8+7+4 = 19
882 → 8+8+2 = 18
847 → 8+4+7 = 19
727 → 7+2+7 = 16
992 → 9+9+2 = 20
676 → 6+7+6 = 19
156 → 1+5+6 = 12
275 → 2+7+5 = 14
725 → 7+2+5 = 14
970 → 9+7+0 = 16

Максимальная сумма цифр = 20

Ожидаемый результат:
992

Реальный вывод программы:
992

Результат корректный.
